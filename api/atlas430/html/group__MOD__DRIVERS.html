Detailed Description

driver for the Maxim DS3234 RTC chip

driver for the TI PGA2311 stereo audio volume control chip

driver for the MCP42xxx dual digital potentiometer chip

Note
the CS signal is managed by the library, but make sure the spi_descriptor is populated correctly
during write transfers, there is no need to set the most significat bit in the address byte to one, the library takes care of it

Petre Rodan
